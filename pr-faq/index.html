Задание 1.

В директории есть файлы program.c; program.h; f.h; f.c. Известно, что в program.h подключен файл f.h. Выберите правильную команду для того, чтобы получить исполняемый файл. Функция main определена в файле program.c. Выберите один ответ:

  • gcc -c program.c f.c
  • gcc -c program.c
  • gcc program.c f.c
  • gcc program.c f.h

Задание 2.

#include
#define PROD<пропуск>

int main() {
    int a = 10;
    printf("%d",PROD(10,4));
    return 0;
}
Что необходимо прописать на месте <пропуск>, чтобы программа выводила 40. Выберите один ответ:
  • a * b
  • (a,b) (a * b)
  • a,b (a*b)
  • Ни один вариант не правильный

Задание 3.

Напишите программу, которая принимает на вход массив целых чисел длины не более 20. В массиве замените все числа на 0, если сумма всех чисел в исходном массиве после него была четной. Например, массив {1 3 1 2 1 3} должен преобразоваться к следующему {0 3 0 0 1 3}. Для последнего элемента сумма не считается
